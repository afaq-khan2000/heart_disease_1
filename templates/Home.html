$ Heart Disease Prediction System

Heart Disease Prediction System

Project Aim

The main purpose of this project is to predict whether a person has a heart disease or not?

Input and Output

Input: Patient
Output: Prediction (Heart Disease Diagnosed / No Heart Disease Diagnosed)

Representation of Input (Patient)

Input (Passenger) is Represented as 4 Input Attributes Input

  • Input Attribute 01– Age (Patient Age)
  • Input Attribute 02 – trestbps value
  • Attribute 03 – Cholestrol Value
  • Input Attribute 04 – FBS value

Representation of Output

Output (Prediction) is Represented as a Single Output Attribute
Output Attribute – Heart Disease

  • Possible Value 01 – Yes
  • Possible Value 02 – No
